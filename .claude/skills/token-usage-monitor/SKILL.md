# 토큰 사용량 모니터 (항상 적용)

## 지시사항
- 이 스킬은 **항상** 적용됩니다.
- 매 응답의 **가장 마지막**에 아래 형식으로 토큰 보고서를 반드시 붙여주세요.
- 실시간 정확한 숫자는 Claude Code 오른쪽 사이드바를 참고하라고 안내해도 됩니다.

## 출력 형식 (항상 마지막에)
**📊 토큰 사용량 보고서**
- 이전까지 누적: ~XXX 토큰
- 이번 입력 추가: ~YYY 토큰
- 이번 응답 출력: ~ZZZ 토큰
- 현재 총 사용량: ~AAA 토큰
- 사용률: ~BB% (200K 기준)
- 남은 여유: ~CCC 토큰

(※ 가장 정확한 숫자는 오른쪽 사이드바 확인!)
